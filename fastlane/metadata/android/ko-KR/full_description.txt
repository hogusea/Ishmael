ISHMAEL Wallet은 Mobick(BMB) 네트워크 사용자를 위한 감시 전용(Watch-only) 지갑입니다.
개인키를 앱에 직접 넣지 않고, 외부 서명 장치와 함께 안전하게 운용하는 PSBT 워크플로우에 맞춰 설계했습니다.

주요 기능
- 감시 전용 지갑 가져오기
  xpub/zpub/descriptor 기반으로 지갑을 가져와 잔액과 거래를 확인할 수 있습니다.

- PSBT 기반 오프라인 서명 흐름
  앱에서 트랜잭션 초안을 만들고, 외부 서명기에서 서명 후 다시 앱으로 가져와 전파할 수 있습니다.

- Mobick 기본 네트워크 설정
  기본 Electrum 서버와 블록 탐색기(`blockchain.mobick.info`)가 Mobick 환경에 맞게 구성되어 있습니다.

- KRW 시세 표시
  원화 환산은 Mobick 티커의 `KRW-BMB` 값을 기준으로 제공됩니다.

- 보안 기능
  앱 잠금, 생체 인증, 로컬 데이터 보호를 지원해 일상 사용 시 보안을 강화합니다.

- 오픈소스
  프로젝트 코드는 공개되어 있으며, 누구나 검토하고 직접 빌드할 수 있습니다.

이 앱은 거래소/수탁형 지갑이 아닙니다.
복구 구문(시드) 보관 및 최종 자산 보호 책임은 사용자에게 있습니다.
